# UX Reference: Futuristic Tech Direction

Use this as the visual default for future UI work in Agenda Workflow.

## Core Mood
- Futuristic, focused, premium, high-trust.
- Feels like a private operations console, not a generic SaaS dashboard.
- Dark surfaces with controlled neon energy.

## Visual Language
- Base: deep navy/near-black solid background.
- Accent energy: blue + violet + pink glows, mostly near top regions.
- Surfaces: layered glass-metal cards with subtle inner highlights.
- Borders: thin, cool-toned, low-contrast lines.
- Shadows: soft depth, no heavy blur clouds.
- Typography: confident, large headings with clear hierarchy.

## Interaction Style
- Motion should be restrained and calm.
- Use subtle hover illumination and slight elevation changes.
- Prioritize readability and scan speed over decoration.

## What To Avoid
- Bright noisy multi-color themes.
- Flat template-like cards.
- Overly dense admin-table layouts.
- Excessive icon clutter or heavy borders.

## Product Principle
- The interface should communicate: command, clarity, discipline.
- Automation should be visible only as polished status signals.
